    [JavaScript]number type, 대입연산자, 산술연산자, 연결연산자

    위치이동
    취소확인

    [number type, 대입연산자, 산술연산자, 연결연산자]

    -number type

    └산술 연산이 가능하다

    └세부적으로 분류하면 int(정수)형과 float(실수) 형으로 분류할 수 있다.

    └정수도 number type!! 실수도 number type!!

    -연산자 우선순위

    └산술연산자 > 대입연산자 (산술연산자가 먼저 동작 후 대입연산자가 동작한다.)

    -문자와 문자을 + 하면?

    └두 문자를 연결해주는 연결연산이 된다.

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Step01_number.html</title>
</head>
<body>
    <script>
       
        //산술 연산
        var num1 = 5+5;
        var num2 = 10.1;

        //number type이 들어 있는 변수명으로도 연산 가능
        var result1 = num1+1;
        
       
        //★실수도 number type! 
        //result엔 number type의 20.1이 들어있다고 말할 수 있다.
        var result2 = num1+num2; 


        //문자와 문자를 + 연산하면 연결연산이 된다.
        var strNum = "999";
        var strNum2 = "1.1";
        var result3 = strNum+strNum2;

    </script>
</body>
</html>
